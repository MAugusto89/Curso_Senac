Exercícios 1 e 2

Exercício 1

  • ALIGN

    • O objetivo do atributo de alinhamento HTML é especificar o alinhamento dos dados e a justificativa do texto em uma célula de uma tabela.

    Elementos suportados

    O atributo de alinhamento HTML suporta os elementos col, colgroup, tbody, td, tfoot, th, thead, tr. O uso do atributo align para qualquer outro elemento HTML foi descontinuado. Você deve usar CSS para eles.

  • VALING

    • O objetivo do atributo valign HTML é definir o alinhamento vertical do conteúdo de uma célula da tabela.

    Elementos suportados

    O atributo valign HTML suporta os elementos col, colgroup, tbody, td, tfoot, th, thead, tr.

  • CELLPADDING e CELLSPACING

    • Existem dois atributos chamados

    cellpadding e cellspacing , que você usará para ajustar o espaço em branco nas células da tabela. O atributo cellspacing define o espaço entre as células da tabela, enquanto o cellpadding representa a distância entre as bordas da célula e o conteúdo dentro de uma célula.

  • COLSPAN e ROWSPAN

    • Você usará o atributo colspan se quiser mesclar duas ou mais colunas em uma única coluna. De maneira semelhante, você usará rowspan se quiser mesclar duas ou mais linhas.

  • WIDTH

    • A propriedade CSS width determina a largura da área de conteúdo de um elemento. A área de conteúdo fica dentro do preenchimento, da borda, e da margem de um elemento.
